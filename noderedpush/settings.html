Push URL

Send a POST request with JSON {"html": "<div>...</div>"} to update the display immediately.

Full URL = your InkyPi base URL + this path (e.g. http://192.168.1.10:8080{{ push_api_path }}).

Using from Node-RED

  1. Use an HTTP Request node: Method = POST, URL = your InkyPi base URL + the path above.
  2. Set msg.payload to { "html": "<div>Your HTML here</div>" }.
  3. Set msg.headers["Content-Type"] = application/json (or use a Function node to build the request).
